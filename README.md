## 🌐 Overview
**CTF-Buddy** is a full-stack, AI-driven assistant designed to act as a mission co-pilot for cybersecurity enthusiasts. Unlike general-purpose LLMs, CTF-Buddy is engineered with a specific system architecture to provide technical exploitation insights, payload analysis, and cryptographic assistance in real-time.



## ✨ Key Features
*   **Neural Interface:** A modern, abstract UI featuring glassmorphic design and a reactive particle background.
*   **Context-Aware Intelligence:** Specialized system prompts to assist with Web Exploitation, Reverse Engineering, and Forensics.
*   **Low-Latency Backend:** Powered by a decoupled Python Flask architecture.
*   **Production-Ready:** Optimized for deployment with secure environment variable management.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (Canvas API).
*   **Backend:** Python 3.x, Flask.
*   **AI Engine:** Google Gemini 2.5 Flash API.
*   **Deployment:** Gunicorn (Production Server).
